# unauthorized-passenger

Unity GAL template prototype for `unauthorized-passenger` / `未许客`.

如果你是第一次从 GitHub 拉项目，先看：

[新手拉取项目指南](GET_PROJECT_GUIDE.md)

## Project Layout

- `Assets/StreamingAssets/GAL/gal_story.json`: story flow, backgrounds, hotspots, languages, portrait library.
- `Assets/StreamingAssets/GAL/Text/`: separated language CSV files.
- `Assets/StreamingAssets/GAL/Portraits/`: standing portrait replacement folder.
- `Assets/StreamingAssets/GAL/Backgrounds/`: current line-art backgrounds.
- `Assets/_Project/Scripts/GalTemplate/`: runtime scripts.
- `Assets/_Project/Docs/GAL_TEMPLATE_EDITING.md`: editing guide for writers and planners.

## Unity Version

Recommended:

```text
Unity 2022.3.62f2
```

Open the project through Unity Hub, then run `Assets/Scenes/SampleScene.unity`.
